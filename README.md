# 💸 Expense Tracker (Python GUI)

A simple and user-friendly **Expense Tracker desktop application** built using Python.  
This application helps users manage daily expenses efficiently with features like adding, updating, deleting, filtering, and visualizing expenses.

---

## 🚀 Features

- ➕ Add new expenses (Date, Category, Amount, Note)
- ✏️ Update existing expense records
- ❌ Delete selected expenses
- 🔍 Filter expenses by category
- 📊 View expense distribution using a pie chart
- 💰 Real-time total expense calculation
- 🧾 Data stored in CSV file for persistence

---

## 🛠️ Technologies Used

- **Python**
- **Tkinter** – GUI development
- **ttk (Themed Tkinter)** – modern widgets
- **CSV** – data storage
- **Matplotlib** – data visualization
- **OS module** – file handling
